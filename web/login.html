Integração com Anchor via SEP-10 para autenticação de wallet O Boleto Guardian não atua como Anchor. Este app consome testanchor.stellar.org — Anchor real na Stellar Testnet — para autenticar sua wallet via SEP-10 (Stellar Web Authentication). A Anchor emite um JWT que comprova o controle da chave privada.
Chave pública derivada

A chave secreta é usada apenas em memória para assinar o desafio da Anchor e nunca é armazenada ou enviada ao servidor.
