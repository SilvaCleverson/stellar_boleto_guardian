Anchor integration via SEP-10 for wallet authentication Boleto Guardian does not act as an Anchor. This app consumes testanchor.stellar.org — a real Anchor on Stellar Testnet — to authenticate your wallet via SEP-10 (Stellar Web Authentication). The Anchor issues a JWT that proves control of the private key.
Derived public key

The secret key is used only in memory to sign the Anchor challenge and is never stored or sent to the server.
